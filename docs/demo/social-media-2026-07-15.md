### Hacker News Style Launch Post

---

**Show HN: Remember Me — AI that remembers non-technical users' style & preferences**

Hey HN,

We just shipped v0.3.0 of Remember Me, a VS Code extension that gives AI persistent memory tailored for non-technical users (PMs, operators, designers, students, founders).

**Problem we solve:**
Every AI chat starts from zero. Users repeatedly re-explain their tone, format preferences, team context, and past decisions. It's exhausting and wastes tokens.

**v0.3.0 highlights:**

- **Template System** — 8 scenarios (PRD, business plan, academic paper, market research, event planning, design doc, tech spec, report). Each template adapts to the user's learned style.
- **Style Consistency Guard** — On document save, automatically compares against historical preferences and flags deviations.
- **Context-Aware Recommendations** — Surfaces relevant past decisions based on current document content using content-sensing algorithms.
- **Memory Version Control** — Keeps last 20 snapshots of memory state. One-click rollback.
- **Semantic Search MVP** — ChromaDB + sentence-transformers for meaning-based retrieval.
- **O(1) Search** — In-memory inverted index for instant lookups.
- **Multi-Provider** — DeepSeek, Qwen (Alibaba), Ernie (Baidu), ChatGLM (Zhipu), Ollama, LM Studio.

**Tech stack:** TypeScript, VS Code Extension API, ChromaDB, sentence-transformers.

We'd love feedback, especially from folks building AI tools for non-technical audiences.
